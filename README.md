# 📱 Tasko

A minimal **React Native To-Do app** with full CRUD functionality, powered by **Convex (BaaS)** and supporting **light & dark themes**.

---

## ✨ Features

- Create, update & delete tasks  
- Convex backend integration  
- Light & Dark mode  
- Real-time updates  

---

## 🛠️ Tech Stack

- React Native  
- Convex
